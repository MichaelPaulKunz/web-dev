Azure AD Migration, Cat Fin, Q3 2024

Situation: API had oauth2 security layer using PingFederate framework. Client had to comply with company-wide migration to Azure AD Framework.

Action: Obtained Azure Client ID and Secret from security team, and registered API with Azure. Worked from Microsoft docs to fetch and include temporary Azure tokens with all API requests from Twilio. Imported jsonwebtoken library into authorizer Lambda function to decode and validate tokens. Included failover that validates PingFederate token if Azure validation fails.

Results: Completed the migration of our API more than one month ahead of the Oct 2024 deadline. Provided client's Salesforce team with roadmap for migration, while allowing them to stay on PingFederate temporarily.

At Rest Call Recording Encryption, Cat Fin, Q3 2023

Situation: Call recordings were encrypted in Twilio and decrypted before storage in S3. Client required them to stay encrypted until playback.

Action: Moved functions to convert audio to buffer, decrypt, and convert back from recordings-processing Lambda that runs when recordings are stored in s3 to get-recordings Lambda that runs when recordings are played. Added recordings-temp S3 bucket to hold decrypted recordings for one day. Added metadata to encrypted recordings.

Results: New recordings remain encrypted until playback. Older, already decrypted recordings are fetched and played same as before.
